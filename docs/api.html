API Reference

Documentacao completa dos endpoints REST do Vectorizer.

POST /vectorize

Vetoriza uma imagem e retorna JSON com o SVG gerado e metadados das regioes.

Parametros (multipart/form-data)

ParametroTipoDefaultDescricao
fileFileobrigatorioImagem (PNG, JPG, WEBP, BMP, HEIC, AVIF)
modestringlineartlineart, icon, vtracer, deep_edges
edge_detectorstringenhanced_cannycanny, hed, enhanced_canny
line_thresholdint170Sensibilidade de deteccao de linhas (0-255)
dilate_radiusint1Raio de dilatacao para fechar gaps nas linhas
min_areaint10Area minima de regiao em pixels
simplifyfloat1.5Tolerancia de simplificacao Douglas-Peucker
max_dimensionint800Dimensao maxima de saida
include_lineartbooltrueIncluir camada de line art no SVG
num_colorsint16Numero de cores para quantizacao (4-32)
vtracer_colormodestringcolorModo VTracer: color ou binary
vtracer_filter_speckleint4VTracer: filtro de ruido
vtracer_color_precisionint6VTracer: precisao de cor
vtracer_layer_differenceint16VTracer: diferenca entre camadas
vtracer_corner_thresholdint60VTracer: threshold de cantos
vtracer_length_thresholdfloat4.0VTracer: threshold de comprimento
vtracer_splice_thresholdint45VTracer: threshold de juncao

Resposta (200 OK)

{
  "svg": "<svg xmlns=...>...</svg>",
  "regions": [
    { "id": 1, "area": 5000, "bbox": [10, 20, 100, 80] }
  ],
  "width": 800,
  "height": 600,
  "processing_time_ms": 245.3
}

POST /vectorize/download

Vetoriza e retorna o SVG como arquivo para download direto.

Parametros

ParametroTipoDefaultDescricao
fileFileobrigatorioImagem de entrada
line_thresholdint170Sensibilidade de linhas
dilate_radiusint1Raio de dilatacao
min_areaint10Area minima de regiao
simplifyfloat1.5Tolerancia de simplificacao
include_lineartbooltrueIncluir line art

Retorna: image/svg+xml com header Content-Disposition: attachment

POST /preprocess/remove-bg

Remove o fundo da imagem usando rembg. Retorna PNG com transparencia.

Parametros

ParametroTipoDescricao
fileFileImagem de entrada

Retorna: image/png

POST /preprocess/upscale

Aumenta a resolucao da imagem por 2x ou 4x.

Parametros

ParametroTipoDefaultDescricao
fileFileobrigatorioImagem de entrada
scaleint2Fator de escala (2 ou 4)

Retorna: image/png

POST /analyze

Analisa o tipo da imagem e retorna configuracoes recomendadas de vetorizacao.

Resposta

{
  "type": "lineart",
  "confidence": 0.92,
  "recommended_mode": "lineart",
  "recommended_settings": { ... },
  "analysis": { ... }
}

POST /extract-text

Extrai texto da imagem usando OCR. Retorna regioes de texto com bounding boxes.

Resposta

{
  "regions": [
    { "text": "Hello", "bbox": [10, 20, 80, 40], "confidence": 0.95 }
  ],
  "count": 1
}

GET /health

Health check do servidor.

Resposta

{ "status": "ok", "version": "2.0.0" }

Erros

CodigoDescricao
400Tipo de arquivo nao suportado ou arquivo muito grande (max 20 MB)
500Erro interno na vetorizacao/processamento
501Dependencia opcional nao instalada (ex: rembg)
