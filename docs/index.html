Pre-processamento

Remocao de fundo, upscale (2x/4x), OCR e analise automatica de tipo de imagem.

Pipeline de Vetorizacao

6 etapas automatizadas transformam pixels em vetores editaveis.

1

Pre-processamento

Resize, denoise bilateral, CLAHE

2

Binarizacao

Deteccao de linhas e bordas

3

Gap Closing

Morfologia para fechar gaps

4

Flood Fill

Connected components para regioes

5

Contornos

Douglas-Peucker + Bezier

6

SVG

Montagem final com regioes + lineart

Quick Start

Rode localmente em minutos.

Backend

# Instalar dependencias e rodar servidor
cd backend
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn server:app --reload --port 8080

Frontend

cd frontend
npm install
npm run dev
# Acesse http://localhost:5151

CLI

python main.py imagem.png
python main.py imagem.png -o resultado.svg
python main.py imagem.png --dilate 4 --threshold 150

Docker

docker build -t vectorizer .
docker run -p 8080:8080 vectorizer

API Reference

Endpoints REST para integrar vetorizacao em suas aplicacoes.

POST /vectorize Vetoriza imagem e retorna JSON com SVG + metadados
POST /vectorize/download Retorna SVG como arquivo para download
POST /preprocess/remove-bg Remove fundo da imagem (retorna PNG)
POST /preprocess/upscale Upscale 2x/4x (retorna PNG)
POST /analyze Analisa tipo de imagem + configuracoes recomendadas
POST /extract-text Extrai texto via OCR
GET /health Health check do servidor
